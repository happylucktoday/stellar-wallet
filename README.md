# Stellar Wallet

One wallet to rule them all. Desktop / Mobile app, payment requests, multisignature and more.

⚠️ Under heavy development 🛠

## Setup

```
npm install
```

## Development

```
npm run dev
```

To run the tests:

```
npm test
```

To run the storybook:

```
npm run storybook
```

## Build

```
npm run build
```
